## ğŸš€ **Quick Start**

### **Prerequisites**
- Docker & Docker Compose
- Node.js 18+ (for GUI)
- Python 3.11+
- Kubernetes cluster (for production deployment)

### **Development Setup**

```bash
# Clone the repository
git clone https://github.com/over7-maker/Advanced-Multi-Agent-Intelligence-System.git
cd Advanced-Multi-Agent-Intelligence-System

# Set up environment
cp .env.example .env
# Edit .env with your API keys and configuration

# Install dependencies
pip install -r requirements.txt
cd frontend && npm install && cd ..

# Run development server
docker-compose up -d

# Access dashboard
open http://localhost:3000
```

### **Production Deployment**

See [PRODUCTION_ROADMAP.md](docs/PRODUCTION_ROADMAP.md) for complete deployment guide.

```bash
# Deploy foundation infrastructure
kubectl apply -f k8s/foundation/

# Deploy intelligence services
kubectl apply -f k8s/intelligence/

# Verify deployment
kubectl get pods -n amas
```
