## ⚠️ Known Issues

### **Current Limitations:**

1. **Stress Test Performance**
   - The system shows degraded performance with 1000+ concurrent requests
   - **Mitigation**: Rate limiting prevents this in normal operation
   - **Fix**: Performance tuning planned for v2.1

2. **Language Detection Edge Cases**
   - Mixed-language issues may be misclassified
   - **Mitigation**: Defaults to English for unclear cases
   - **Fix**: ML-based detection in roadmap

3. **Cache Size Growth**
   - SQLite database grows over time without cleanup
   - **Mitigation**: 24-hour TTL limits growth
   - **Fix**: Automated cleanup job in development

### **Not Issues (By Design):**
- Single workflow instance (GitHub Actions limitation)
- 5-minute timeout for long operations
- Template responses for unrecognized patterns
